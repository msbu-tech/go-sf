Service Frame generator for Dpp in Go. <https://msbu-tech.github.io/go-sf/>

## Install

```
go get github.com/msbu-tech/go-sf
```

## Usage

```
go-sf -new -name=doctor \
      -author=msbu \
      -template=template/service \
      -output=/path/to/output
```
